# Joystick ile LED Kontrolü

Bu proje, bir Arduino ve Joystick modülü kullanarak LED'lerin parlaklığını veya yönünü kontrol etmeyi sağlar. Mekatronik mühendisliği çalışmaları kapsamında geliştirilmiştir.

## İçerik
* **Kod:** Arduino için yazılmış kontrol yazılımı.
* **Görseller:** Devre kurulumuna ait fotoğraflar.
* **Video:** Sistemin çalışma performansını gösteren kısa video.
